# Programação Competitiva

Este repositório armazena minhas soluções de problemas de programação competitiva.

## Contexto

Comecei a estudar programação competitiva por causa da disciplina da faculdade chamada TALP (Tópicos Avançados em Linguagem de Programação). O foco inicial da matéria era a preparação para a Maratona de Programação da SBC de 2026.

## Objetivos

Apesar de ter começado por causa da disciplina, peguei gosto pela área e vou continuar os treinos. Meus principais objetivos agora são:

* Melhorar minhas habilidades gerais de programação.
* Aprofundar conhecimentos em Estruturas de Dados e Algoritmos.
* Desenvolver raciocínio rápido e pensamento crítico para a resolução de problemas.

## Tecnologias Utilizadas

* Linguagem: C++
* Plataforma principal: Codeforces
* Meu perfil: **[Augusto-Cesar](https://codeforces.com/profile/Augusto-Cesar)**

## Estrutura do Repositório

As pastas estão organizadas categorizando as soluções de acordo com o nível de dificuldade (rating) e os títulos do Codeforces:

* `800-900` (Newbie)
* `1000-1100` (Pupil)
* `1200-1300` (Apprentice)
* `1400-1500` (Specialist)
* E assim por diante, acompanhando a evolução dos treinos.
